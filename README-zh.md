# IoIo Tools - JSON 查看器和编辑器

[English Version](README.md)

一个功能强大的在线 JSON 查看器和编辑器工具，帮助你格式化、验证和转换 JSON 数据。

## 功能特性

- 🚀 多种视图

  - 树状视图：层级展示 JSON 数据
  - JSON 视图：带语法高亮的原始 JSON
  - 对比视图：比较 JSON 差异
  - 转换视图：在不同格式之间转换

- 📑 多标签页支持

  - 多文档管理
  - 自定义标签页名称
  - 跨会话持久化标签页

- 🔄 格式转换

  - JSON
  - YAML
  - TOML
  - JSON5
  - TypeScript
  - 表单数据
  - 查询字符串

- 🌐 国际化

  - 英文
  - 简体中文
  - 更多语言即将推出

- 🎨 主题支持
  - 亮色模式
  - 暗色模式
  - 自动模式（跟随系统偏好）

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产环境
pnpm build

# 启动生产环境服务器
pnpm start
```
